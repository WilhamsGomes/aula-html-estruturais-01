Linguagem de marcação - HTML

    		HTML abreviação para a expressão inglesa HyperText Markup Language,
    		que significa: "Linguagem de Marcação de Hipertexto" é uma linguagem de 
    		marcação utilizada na construção de páginas na Web. Documentos HTML 
    		podem ser interpretados por navegadores.

class=”…“

Atribui uma classe ao elemento (uma classe pode ser utilizada para um ou mais elementos);

id=”…“

Atribui um id ao elemento (um id deve ser único, ou seja atribuído a um único elemento);
